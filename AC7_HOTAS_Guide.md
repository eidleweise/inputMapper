# Ace Combat 7 - HOTAS Input.ini Translation Guide

This guide maps the development terms found inside the `Input.ini` configuration file to standard gamepad layout logic and recommended configurations for a flight stick & throttle setup (such as the VKB Gladiator NXT/EVO and Thrustmaster TWCS).

| **`Flight_Roll`** | Banks wings left / right | Left Stick (Left / Right) | Gladiator Joystick (X-Axis) |
